# Проект: AI Books - Персонализированные книги с ИИ

## 📋 Описание проекта

Платформа для создания персонализированных развивающих книг для детей и подростков с использованием искусственного интеллекта. Главный герой книги - сам ребёнок или близкий человек.

---

## 🎯 Целевые аудитории

### Дети 0-7 лет
**Жанры:**
- Алфавит (гласные/согласные)
- Цифры (0-10, 10-100)
- Учимся читать (слоги)
- Гигиена
- Правила поведения/Этикет
- Учим животных
- Прописи
- Разновидности фигур
- Русские народные сказки
- Учим цвета
- Национальности
- Изучение подводного мира
- Спасатель в больнице

**Сопутствующие герои:**
- Динозавры, Принцессы, Супергерои, Единороги, Инопланетяне
- Выдуманные герои, Животные, Феи, Пираты, Доктор

**Главные герои:**
- Родители, Брат, Сестра, Друзья, Родственники

**Места обитания:**
- Лес, Сад, Улица, Цирк, Парк, Озеро, Река, Набережная
- Комната, Двор, Садик, Школа, Зоопарк, Замок, Пустыня
- Море, Луг, Поле, Баня, Дворец, Космос, Город, Мегаполис
- Детский сад, Корабль, Футбольное поле, Океан, Больница

---

### Дети 7-12 лет
**Жанры:**
- Виды машин и база про машины
- Первые шаги в моду
- Как бороться с буллингом
- Первые шаги в мир брендов
- Первое знакомство с косметикой
- Как правильно худеть
- Шахматы, Вязание, Оригами, Бисер
- Композиции, Как правильно рисовать
- Опыты, Сборник цитат
- Правила дорожного движения
- Изучение космоса

---

### Подростки 12-18 лет
**Жанры:**
- **Любовь** - книга о истории отношений (фото + описание знакомства)
- **Близкий человек** - для лучших друзей/подруг
- **Поздравления** - мини-истории знакомства с друзьями
- **Родственники** - волшебное дерево семьи
- **Подарок на 18 лет** - книга с 18 фотографиями
- **Знакомство со звездой** - книга с кумиром как другом
- Как пользоваться Excel, Word и т.д.
- Идеи напитков и рецепты (кофе, матча)
- Рецепты
- Изучение языков по личной программе

## 👥 Роли пользователей

### Гость
- Не зарегистрированный пользователь
- Может просматривать сайт
- Может выбирать просмотр ячеек и страниц

### Клиент
- Зарегистрированный пользователь
- **Возможности:**
  - Просмотр сайта, ячеек, страниц
  - Оформление заказа
  - Просмотр статуса заказа
  - Выбор жанра, цвета, музыки
  - Настройка количества страниц, формата
  - Ввод личных данных (имя, фамилия, пол, адрес, фото)
  - Добавление ячеек (друзья, родители)
  - Выбор места сказки
  - Работа с черновиками
  - Поддержка, отзывы, FAQ

### Администратор
- Контроль заказов с обратной стороны
- Просмотр сведений о типографии (для бумажных версий)
- Ответы на вопросы в поддержке (если бот не справляется)

## 🔌 Внешние сервисы

### Подключения:
- 💳 Оплата
- 📱 SMS
- 📧 Почта
- 📊 CRM
- 🚚 Доставка

---

## ⚠️ Ошибки и поддержка

### Типичные проблемы:
- Неверный пароль
- Товар закончился
- Обрыв связи
- Неверные данные формы

### Что видит пользователь:
- Понятный текст без технических кодов
- Кнопка «Повторить» или «В поддержку»

### Что запоминает система:
- Сохраняет черновик
- Записывает лог ошибки
- Не списывает деньги дважды
