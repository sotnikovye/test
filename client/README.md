# AI Books - Фронтенд

Персонализированные развивающие книги с искусственным интеллектом.

## 🚀 Быстрый старт

### Установка зависимостей
```bash
cd client
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```
Приложение будет доступно по адресу: http://localhost:3000

### Сборка для продакшена
```bash
npm run build
```

### Предварительный просмотр продакшен сборки
```bash
npm run preview
```

## 🎨 Цветовые схемы

Проект использует 3 цветовые схемы согласно ТЗ:

1. **Мятный/Персиковый**: `#B4E7D1` / `#FFDAB9`
2. **Сливочный/Коричневый/Золотой**: `#FFFDD0` / `#8B4513` / `#FFD700`
3. **Малиновый/Фисташковый**: `#DC143C` / `#93C572`

## 🛠 Технологии

- **React 19** - UI библиотека
- **Vite** - Сборщик
- **React Router DOM** - Роутинг
- **Tailwind CSS 4** - Стилизация
- **Axios** - HTTP клиент
- **Lucide React** - Иконки

## 📱 Страницы и функционал

### Публичные страницы
- **Главная** - Презентация продукта, возрастные группы, преимущества
- **Каталог** - Фильтрация по возрасту, жанру, поиск
- **Конструктор книг** - 4-шаговый мастер создания книги

### Требуется аутентификация
- **Личный кабинет** - Профиль, статистика
- **Заказы** - История заказов, статусы
- **Черновики** - Сохраненные черновики книг
- **Поддержка** - FAQ, форма обращения

### Администрирование
- **Админ-панель** - Статистика, заказы, обращения

## 🔐 Аутентификация

Компонент `AuthContext` предоставляет:
- Регистрацию (email, телефон, пароль от 6 символов)
- Вход/выход
- Проверку прав (клиент/админ)
- Сохранение токена в localStorage

## 🌐 API Интеграция

Базовый URL API: `http://localhost:5000/api`

Настройте переменную окружения:
```bash
# .env
VITE_API_URL=http://your-api.com/api
```

## 📦 Основные возможности

### Для детей 0-7 лет
- Алфавит, цифры, учимся читать
- Гигиена, правила поведения
- Животные, прописи, фигуры, цвета

- Машины, мода, бренды

- Любовные истории
- Поздравления друзьям
- Рецепты, языки, IT навыки

- Любимая песня в PDF
- ИИ поздравления (+1000₽)
- ИИ мультики с лицом ребенка (+1500₽)
